1 Переменные

1.1 Объявление переменных

var a

1.1.1 Объявление с инициализацией

var b = 10

1.2 Имена переменных

Имя переменной может начинаться только с латинской буквы или нижнего подчеркивания: 
var a1
var _A2
var 2a <- Ошибка!
var "A" <- Ошибка!

1.3 Тип переменных

1.3.1 Тип переменных указывать не требуется

1.3.2 Тип переменных задается во время работы программы и зависит от типа значения RHS (right-hand side) выражения.

1.3.3 Тип переменных задается только единожды в момент объявления переменной и более меняться не может.

1.4 Присваивание значений переменным

Переменным могут присваиваться значения переменных, констант, вызовов функций, при чем тип RHS значения должен иметь тот же тип, что и тип LHS. 
a = 5 + 6
b = a
c = add(a, b)

2 Числовые выражения

Язык предоставляет следующие операции для работы с числовыми выражениями:

2.1 Операции типа сложения

5 + 5 // 10
4 - 7 // -3
2 + (- 10) // -8

2.1.1 Операндами операции сложения являются только числовые выражения

2.2 Операции типа умножения

var a = 5 * 5 // 25
var b = a / 50 // 0.5

2.2.1 Операндами операции умножения являются только числовые выражения

3 Булевые выражения

Язык предоставляет следующие операции для работы с булевыми выражениями:

3.1 Операции сравнения больше/меньше

var a = 5 < 10 // true
-1 > 0 // false

3.1.1 Операция сравнения больше/меньше принимает только числовые выражения

3.2 Операция сравнения эквивалентно

5 == 5 // true
false == true // false

3.2.1 Операция сравнения эквивалентно принимает как числовые, так и булевые выражения

3.3 Логическое ИЛИ

(5 > 10) || true // false

3.3.1 Операция принимает только булевые выражения

3.4 Логическое И

(10 == 10) && true // true

3.4.1 Операция принимает только булевые выражения

4 Циклы

4.1 For цикл

4.1.1 Синтаксис:

for ( <initialization>; <condition>; <increment expr> ) {
   <statement>
}

4.1.2 Наличие initialization необязательно

var i = 0
for ( ; i < 10; i = i + 1) {}

4.1.3 Переменная, объявлененаая в initialization, уничтожается после завершения цикла

4.1.4 Любая переменная, которая была объявлена внутри тела цикла, уничтожается после выхода из цикла

4.1.5 В качестве condition может использоваться только выражение, имеющее тип bool

4.1.6 Бесконечный цикл

for ( ; ; ) {}

4.1.7 Break инструкция

Break инструкция останавливает выполение цикла 
for (var i = 0; i < 10; i = i + 1) {
    ...
    if (flag) {
        break
    }
}

5 Условные инструкции

5.1 If Statement

if (<condition>) {
    <statement>
}

5.2 Else Statement:

Else инструкция позволяет добавить еще один блок, который обязательно выполнится, если предшествующее ему условие ложно 
if (<condition>) {
    <statement>
} else {
    <statement>
}

5.3 Else If Statement

Else if инструкция позволяет добавить еще одно условие с отдельным блоком для проверки, если первое ложно 
if (<condition>) {
    <statement>
} else if (<condition2>) {
    <statement>
} else {
    <statement>
}

5.3 Наличие condition обязательно

5.4 Тело может быть пустым

5.5 Любая переменная, обяъвленная внутри тела if инструкции будет уничтожена после выхода из инструкции

5.6 В качестве condition может использоваться только выражение, имеющее тип bool

6 Функции

6.1 Объявление функций

func <func_type> <function_name>(<params>) {
    <statement>
    return <expr>
}

6.2 Тип возвращаемого значения функции

Тип возвращаемого значения функции <func_type> может принимать следующие значения:

  • int
  • bool
  • void

6.3 Имена функций

Имена функций <function_name> соблюдает те же правила, что и имена переменных

6.3.1 Имена функций не конфликтуют с именами переменных, то есть одно и тоже имя может использовать как для переменной, так и для функции

6.4 Параметры функций

Параметрами функций могут являться только переменные:

func add(var a, var b) {}

func add(5, 6) {} <- Ошибка!

func add(mul(), var b) {} <- Ошибка!

6.4.1 Аргументы в функции передаются только по значению

6.5 Return инструкция

6.5.1 Return в int, bool функциях

В функциях, тип возвращаемого значения которых объявлен как int или bool, return функция должна присутствовать обязательно

func int add(var a, var b) {
    return a + b
}

        return a
    } else {
        return b
    }

Недопустимо:

func bool get_bool(var a) {
    if (a > 0) {
        return true
    } else {
        ...
    }
}

6.5.2 Return в void функциях

В функциях, тип возвращаемого значения которых объявлен как void, return функция может отсутствовать

func void set_val() {
    a = 5
}
